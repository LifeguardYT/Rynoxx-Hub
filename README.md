Raynoxx Hub
=============

Script Hub for Roblox

This repository contains a Roblox Lua script `Raynoxx Hub.lua` (a Rayfield-based UI with various utilities).

Usage
-
- Place `Raynoxx Hub.lua` into your Roblox executor of choice and run it.

Repository
-
- Initial commit adds the script and basic repo files.

To push to GitHub (after confirming remote creation):

```powershell
cd path\to\raynoxx-hub
git remote add origin <remote-url>
git branch -M main
git push -u origin main
```
